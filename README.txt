# Sudoku
Sudoku Game created by me and my college friend Krzysztof,
Game was written in Java in IntelliJ environment

What you can do in this app:
> play sudoku in 3 different levels
> change language to Polish, English or Russian
> check your mistakes during the game
> save your board to continue it later by reading saved file in menu

In console:
Logs from your actions in this app

What you cannot do:
Write symbols different than numbers in sudoku fields ;)

VM options to open this project in intellij
--module-path YOUR_PATH\javafx-sdk-15.0.1\lib --add-modules=javafx.controls,javafx.fxml

GUI was created in scene builder
